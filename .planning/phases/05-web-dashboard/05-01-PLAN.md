    1. `get_groups_with_summary(db: Session) -> list[dict]`:
       - Query todos os RouteGroup (db.query(RouteGroup).all())
       - Para cada grupo, buscar snapshot mais recente (ORDER BY collected_at DESC, LIMIT 1) para determinar o ciclo mais recente
       - Buscar o snapshot mais barato daquele ciclo (mesmo collected_at, ORDER BY price ASC, LIMIT 1)
       - Buscar sinal mais urgente nas ultimas 12h: query DetectedSignal WHERE route_group_id == group.id AND detected_at >= utcnow() - 12h, ORDER BY CASE urgency (MAXIMA=3, ALTA=2, MEDIA=1) DESC, LIMIT 1
       - Usar `func.case` do SQLAlchemy para ordenar por urgencia: `func.case((DetectedSignal.urgency == "MAXIMA", 3), (DetectedSignal.urgency == "ALTA", 2), (DetectedSignal.urgency == "MEDIA", 1), else_=0)`
       - Retornar list de dicts: {"group": RouteGroup, "cheapest_snapshot": FlightSnapshot|None, "signal": DetectedSignal|None}

       - Calcular cutoff = datetime.utcnow() - timedelta(days=days)
       - Encontrar rota mais barata: GROUP BY origin, destination, ORDER BY func.min(FlightSnapshot.price) ASC, LIMIT 1
       - Se nao encontrar rota, retornar {"labels": [], "prices": [], "route": ""}
       - Buscar todos os snapshots dessa rota com collected_at >= cutoff, ORDER BY collected_at ASC
       - Retornar {"labels": [s.collected_at.strftime("%d/%m %Hh") for s], "prices": [s.price for s], "route": f"{origin} -> {destination}"}
