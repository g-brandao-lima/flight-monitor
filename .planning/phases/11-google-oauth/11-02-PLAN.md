1. Criar app/auth/oauth.py (per D-03, usar Authlib):
```python
from authlib.integrations.starlette_client import OAuth
from app.config import settings

oauth = OAuth()
oauth.register(
    name="google",
    client_id=settings.google_client_id,
    client_secret=settings.google_client_secret,
    server_metadata_url="https://accounts.google.com/.well-known/openid-configuration",
    client_kwargs={"scope": "openid email profile"},
)
```

2. Criar app/auth/routes.py com 3 rotas:
- GET /auth/login: `await oauth.google.authorize_redirect(request, redirect_uri)` (per D-01, redirect_uri aponta para /auth/callback)
- GET /auth/callback (name="auth_callback"): try/except em authorize_access_token. Sucesso: find_or_create User por google_id, seta request.session["user_id"] = user.id, redireciona para /dashboard (per D-01). Erro: redireciona para /?msg=login_erro (per D-07). Se usuario cancelou (OAuthError ou query param error), redirecionar para /?msg=login_cancelado.
- GET /auth/logout: request.session.clear(), redireciona para / (per D-03)

3. Criar app/auth/middleware.py (per D-05):
```python
PUBLIC_PATHS = frozenset({"/", "/auth/login", "/auth/callback", "/auth/logout"})
PUBLIC_PREFIXES = ("/auth/", "/static/", "/api/airports/")
```
